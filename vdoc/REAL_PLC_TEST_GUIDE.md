# 真实 PLC 模式测试指南

## 测试前准备

### 1. 修改配置文件

编辑 `.env` 文件，设置为真实 PLC 模式：

```env
USE_MOCK_DATA=false
ENABLE_POLLING=true

# PLC 配置
PLC_IP=192.168.50.224
PLC_RACK=0
PLC_SLOT=1
PLC_TIMEOUT=5000
```

### 2. 确保 PLC 可访问

```powershell
# 测试 PLC 网络连接
ping 192.168.50.224
```

### 3. 确保 InfluxDB 运行

```powershell
# 检查 InfluxDB 服务状态
curl http://localhost:8086/health
```

## 测试验证点

### 1. 电表数据验证

**电压**：
- 公式：`raw × 0.1`
- 单位：V
- 合理范围：300V - 450V

**电流**：
- 公式：`raw × 0.001 × 20`
- 单位：A
- 合理范围：0A - 200A

**功率**：
- 公式：`raw × 2`
- 单位：kW
- 合理范围：0kW - 100kW

**能耗**：
- 公式：`raw × 2`
- 单位：kWh
- 合理范围：0kWh - 无上限

### 2. 压力数据验证

**压力**：
- 公式：`raw × 1.0`（直接使用原始值）
- 单位：kPa
- 合理范围：0kPa - 1000kPa

### 3. 振动数据验证

**位移**：
- 单位：μm
- 合理范围：0μm - 100μm

**频率**：
- 单位：Hz
- 合理范围：0Hz - 100Hz

**速度**：
- 单位：mm/s
- 合理范围：0mm/s - 50mm/s

## 常见问题排查

### 1. PLC 连接失败

**错误信息**：`PLC 连接失败`

**排查步骤**：
1. 检查网络连接：`ping 192.168.50.224`
2. 检查 PLC IP 配置是否正确
3. 检查 PLC 是否开机
4. 检查防火墙是否阻止连接

### 2. 数据读取失败

**错误信息**：`数据读取失败，长度: 0`

**排查步骤**：
1. 检查 DB 块号是否正确（DB2）
2. 检查 DB 块大小是否正确（1034 字节）
3. 检查 PLC 程序是否运行
4. 检查 DB 块是否存在

### 3. 数据解析失败

**错误信息**：`数据解析失败`

**排查步骤**：
1. 检查配置文件是否正确（`configs/config_waterpump_db2.yaml`）
2. 检查模块配置是否正确（`configs/plc_modules.yaml`）
3. 查看详细错误日志

### 4. 数据转换失败

**错误信息**：`数据转换失败`

**排查步骤**：
1. 检查转换器配置是否正确
2. 检查原始数据是否有效
3. 查看详细错误日志

### 5. InfluxDB 写入失败

**错误信息**：`InfluxDB 不可用`

**排查步骤**：
1. 检查 InfluxDB 服务是否运行
2. 检查 InfluxDB 配置是否正确（`.env` 文件）
3. 检查 Token 是否有效
4. 检查 Bucket 是否存在

## 测试报告解读

### 通过率 100%

表示所有测试项目都通过，系统运行正常。

### 通过率 < 100%

需要查看具体失败的测试项，根据错误信息进行排查。

### 数据完整性测试失败

表示某些设备的数据缺失或字段不完整，需要检查：
1. PLC 程序是否正常
2. 配置文件是否正确
3. 数据解析逻辑是否正确

### 数据合理性测试失败

表示数据超出合理范围，可能原因：
1. 传感器故障
2. 转换系数错误
3. PLC 数据异常

## 测试脚本扩展

如需添加更多测试项，可以在 `test_real_plc_integration.py` 中添加新的测试方法：

```python
async def test_custom_logic(self) -> TestResult:
    """自定义测试逻辑"""
    result = TestResult("自定义测试")
    start_time = time.time()
    
    try:
        # 测试逻辑
        result.passed = True
        result.message = "测试通过"
    except Exception as e:
        result.message = f"测试失败: {e}"
    
    result.duration = time.time() - start_time
    return result
```

然后在 `run_all_tests()` 中调用：

```python
result = await self.test_custom_logic()
self.add_result(result)
```
